## For running the project locally 

### Clone Repository

### Download MongoDB
https://docs.mongodb.com/manual/administration/install-community/

### Run Backend
`npm install`<br />
`npm start`


### Run Frontend
`cd frontend` <br />
`npm install` <br />
`npm start`
